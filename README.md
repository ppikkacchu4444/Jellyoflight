# Jelly of Light

## 🌌 Overview
Save the Jelly World from the encroaching darkness.  
Dive into a visually rich and immersive jelly universe, designed for short yet impactful 10-minute play sessions.

## 🎮 Genre
2D Casual Platformer

## 🕹 Controls
- W / A / S / D – Move  
- Space – Jump  
- Shift – Attack  
- F – Interact  

## ⚡ Core Gameplay
- Stage Exploration  
- Level Interaction  
- Responsive Movement  
- Simple Combat System  

## 🛠 Tech Stack
- Unity 6000.3.9f1  
- C#  
- Gemini, Claude
